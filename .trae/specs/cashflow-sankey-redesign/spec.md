# 重新绘制现金流桑基图规格

## Why
当前现金流量桑基图只显示活动类型（经营/投资/筹资）的流入流出，无法直观展示：
1. 具体的资金来源（哪些收入分类）
2. 具体的资金用途（哪些支出分类）
3. 具体的现金账户（哪些账户在参与现金流）

## What Changes
重新设计桑基图结构为：**资金来源 → 现金账户 → 资金用途**

### 新桑基图结构

```
资金来源              现金账户              资金用途
┌─────────┐         ┌────────┐         ┌─────────┐
│工资收入  │────────▶│        │────────▶│餐饮支出  │
├─────────┤         │ 支付宝 │         ├─────────┤
│投资收益  │────────▶│        │────────▶│交通支出  │
├─────────┤         ├────────┤         ├─────────┤
│银行转入  │────────▶│ 微信   │────────▶│购物支出  │
├─────────┤         ├────────┤         ├─────────┤
│...      │────────▶│ 银行卡 │────────▶│银行转出  │
└─────────┘         └────────┘         └─────────┘
```

### 节点定义

| 层级 | 节点类型 | 数据来源 |
|------|---------|---------|
| 左侧 | 资金来源 | 收入分类（现金流入）、非现金账户（转账转入） |
| 中间 | 现金账户 | 标记为现金等价物的账户 |
| 右侧 | 资金用途 | 支出分类（现金流出）、非现金账户（转账转出） |

### 链接定义

| 链接类型 | source | target | value |
|---------|--------|--------|-------|
| 收入流入 | 收入分类名 | 现金账户名 | 收入金额 |
| 转账转入 | 非现金账户名 | 现金账户名 | 转账金额 |
| 支出流出 | 现金账户名 | 支出分类名 | 支出金额 |
| 转账转出 | 现金账户名 | 非现金账户名 | 转账金额 |

## Impact

### 后端修改
- `backend/src/routes/report.ts`: 修改现金流量API，返回按分类和账户分组的详细数据

### 前端修改
- `frontend/src/pages/Reports.tsx`: 修改桑基图数据结构

## ADDED Requirements

### Requirement: 桑基图数据结构
后端API应返回以下数据结构：

```typescript
interface CashFlowSankeyData {
  nodes: Array<{ name: string; category?: string }>
  links: Array<{ source: string; target: string; value: number }>
}
```

### Requirement: 节点分类
节点应包含 category 属性用于着色：
- `source`: 资金来源节点（绿色系）
- `cash`: 现金账户节点（蓝色系）
- `target`: 资金用途节点（红色系）

### Requirement: 数据过滤
- 只显示金额大于0的节点和链接
- 合并金额较小的分类（如小于总额1%）
